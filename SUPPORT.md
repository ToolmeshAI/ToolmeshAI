# Support

This repository is the profile hub for ToolmeshAI. Support is best routed to the repository that owns the work you are asking about.

## Where To Ask

- Repo-specific bugs or feature ideas: use the issue tracker on the relevant repository.
- Missing docs or unclear examples: open an issue in the relevant repository and link the section that needs work.
- Profile-level suggestions for this page: use this repository when issue tracking is available.
- Security concerns: follow [`SECURITY.md`](./SECURITY.md).

## What To Expect

- Support is best-effort.
- Early-stage repositories may change quickly and may not maintain older examples.
- The focus is on improving public documentation and reusable patterns rather than private one-off support.
