# ToolmeshAI Profile Launch Notes v0.1.0

Date: 2026-04-15

## Summary

`v0.1.0` establishes this repository as the public profile hub for ToolmeshAI. The goal is not to list every experiment. The goal is to present a smaller set of connected repositories that are actually being sharpened in public.

## What Ships In This Launch

- A concise profile README in English with a Chinese mirror
- Baseline community docs for support, security, and conduct
- A release note trail starting from this launch
- A `now` page to make current priorities legible without bloating the README

## Portfolio At Launch

- [`mcp-saas-foundry`](https://github.com/ToolmeshAI/mcp-saas-foundry): flagship blueprint for SaaS-oriented MCP repositories
- [`safe-mcp-config`](https://github.com/ToolmeshAI/safe-mcp-config): safety check for obvious MCP configuration mistakes
- [`docs-to-context`](https://github.com/ToolmeshAI/docs-to-context): tooling for turning source material into clean AI context
- [`awesome-mcp-workflows`](https://github.com/ToolmeshAI/awesome-mcp-workflows): bilingual list of practical MCP workflows and patterns
- [`browser-agent-recipes`](https://github.com/ToolmeshAI/browser-agent-recipes): short, reusable browser automation recipes for agent workflows

## What To Expect Next

- More disciplined README and docs polish across the active repositories
- Small public releases that improve real usage, not empty scaffolding
- Slow expansion of the portfolio, only when a new repository clearly strengthens the set
