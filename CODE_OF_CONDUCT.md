# Code of Conduct

ToolmeshAI projects aim to be practical, respectful, and useful in public. Participation in this repository and related project spaces means following these standards.

## Expected Behavior

- Be respectful, direct, and technically honest.
- Assume good intent first, then discuss disagreements with evidence.
- Keep feedback specific to the work.
- Share context that helps other contributors reproduce or understand a problem.

## Unacceptable Behavior

- Harassment, hate speech, discrimination, or intimidation
- Personal attacks, dogpiling, or sustained hostility
- Publishing private information without permission
- Spam, trolling, or deliberately disruptive behavior

## Scope

This applies to repository issues, pull requests, discussions, release threads, and linked project spaces used for collaboration.

## Reporting

If you experience or witness a problem, contact the maintainers through the least public channel available. If no direct contact path is available yet, open a neutral issue requesting maintainer contact without posting sensitive details.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this code of conduct and may limit participation when needed to protect the project space.
